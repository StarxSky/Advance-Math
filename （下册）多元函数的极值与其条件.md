# 多元函数的极值定义（以二元函数为例，且以下定理适用于多元函数）

定义：
* **极大值**：如果函数$ z= f(x, \space y) $在定义域$D$中的某个点$p_{0}(x, \space y) \in D$( 点$P_{0}(x_{0}, \space y_{0}) $为点集$D$的内点 ) 的某个邻域$ U(p_{0}) \subset D$ 中任取异于点$p_{0}(x_{0}, \space y_{0}) $的点$ p(x, \space y) $时有: $$ f(x, \space y) < f(x_{0}, y_{0})$$成立，则该点$p_{0}(x_{0}, \space y_{0}) $称为极大值点，且，极大值为$ f(x_{0}, \space y_{0}) $

* **极小值**：如果函数$ z= f(x, \space y) $在定义域$D$中的某个点$p_{0}(x, \space y) \in D$( 点$P_{0}(x_{0}, \space y_{0}) $为点集$D$的内点 ) 的某个邻域$ U(p_{0}) \subset D$ 中任取异于点$p_{0}(x_{0}, \space y_{0}) $的点$ p(x, \space y) $时有: $$ f(x, \space y) >f(x_{0}, y_{0})$$成立，则该点$p_{0}(x_{0}, \space y_{0}) $称为极小值点，且，极小值为$ f(x_{0}, \space y_{0}) $


## （函数具有极值的必要条件）
定理： 函数$z = f(x, \space y) $ 在点$(x_{0}, \space y_{0})$处具有偏导数，且在点$ (x_{0}, \space y_{0}) $具有极值，则，有：
$$ f_{x}(x_{0}, \space y_{0}) = \frac{ \partial{f} }{\partial{x}} |_{(x_{0}, \space y_{0})} = 0 \space ; \space f_{y}(x_{0}, \space y_{0}) = \frac{ \partial{f} }{\partial{y}} |_{(x_{0}, \space y_{0})} = 0$$

## （函数具有极值的充分条件）
定理： 函数$z = f(x, \space y) $ 在点$ (x_{0}, \space y_{0}) $的某邻域内连续且有一阶二阶连续的偏导数,同时，$f_{x}(x_{0}, \space y_{0}) = 0 $,$ \space f_{y}(x_{0}, \space y_{0}) = 0 $则如果令：$$ f_{xx}(x_{0}, \space y_{0} ) = A \space; \space f_{xy}(x_{0}, \space y_{0}) = B \space ;  \space f_{yy}(x_{0}, \space y_{0}) = C  $$ 
* 如果 $ AC -B^2 > 0 $则，当$ A > 0$时有极小值， 如果$ A < 0$时有极大值
* 如果$ AC -B^2 = 0 $时，可能村子极值也可能没有极值，根据实际情况进行讨论分析
* $ AC -B^2 < 0 $时，没有极值
