# 对弧长的曲线积分与对坐标的曲线积分之间的联系

设有向线段弧$ L$由参数方程$$ x = x(t) \\ y = y(t) $$给出，起点$ A$终点为$B$对应的参数:$$ A = \alpha \\ B = \beta$$ 函数$ x(t)$与$ y(t)$在区间$ t \in [\alpha, \beta]$上连续，同时存在一阶连续导数$ x^{'}(t)$与$ y^{'}(t)$满足：$$ x^{'2}(t) + y^{'2}(t) \neq 0$$因为函数$ Q(x, y), P(x, y)$ 在曲线弧$ L$ 上连续所以，根据对坐标的曲线积分计算公式：$$ \int_{L} P(x,y) dx + Q(x, y) dy = \int_{\alpha}^{
\beta} P[x(t), y(t)] x^{'}(t)dt + \int_{\alpha}^{\beta} Q[x(t), y(t)] y^{'}(t) dt$$同时，曲线弧$ L$ 在点$M(x(t), y(t))$处的切向量为：$$ d\mathbf{r} = x^{'}(t) i + y^{'}(t)j$$其方向余弦为:$$ cos \alpha = \frac{x^{'}(t)}{\sqrt{x^{'2}(t) +  y^{'2}(t)}} \\ \space \\ cos \beta = \frac{y^{'}(t)}{\sqrt{x^{'2}(t) + y^{'2}(t)} }$$为了将坐标轴上的分量函数$ Q(x, y), \space P(x, y)$转换为沿着曲线弧$ L$在点$ M(x(t), y(t))$处的切线向量方向的量做一下方法：$$ \int_{L}[ P(x, y)cos\alpha+ Q(x, y) cos\beta ] ds$$ 结合对弧长的曲线积分计算方法得到：
$$ \int_{L} \{ P(x(t), y(t)) \frac{x^{'}(t)}{ \sqrt{x^{'2}(t) 
+ y^{'2}(t)}} + Q(x(t),y(t)) \frac{y^{'}(t)}{\sqrt{x^{'2}(t) + y^{'2}(t)}} \} \sqrt{x^{'2}(t) + y^{'2}(t)} dt \\\space  \\ = \int_{\alpha}^{\beta}\{P[x(t), y(t)] x^{'}(t) + Q[x(t), y(t)] y^{'}(t) \} dt $$
因此，平面上的曲线弧的两类积分之间具有的关系可以总结如下 $$ \boxed{\int_{L} P dx + Q dy = \int_{L} (Pcos\alpha + Qcos\beta) ds}$$
其中: $$ ds = \sqrt{x^{'2}(t) + y^{'2}(t)} dt$$
其中$ \alpha(x, y), \beta(x, y)$ 为曲线弧$ L$ 在点$ M(x, y)$处的切向量的方向角该方法同样可以推广到空间中的有向曲线弧$ \Lambda$上: $$\boxed{ \int_{\Lambda} Pdx + Qdy + Rdz = \int_{\Lambda} \{{P cos\alpha + Q cos\beta + R cos\gamma} \}ds}$$

* 向量方式： $$ \boxed{\int_{L} \bar{A} \cdot \bar{dr} = \int_{L} \bar{A} \cdot \bar{\tau} ds }$$其中的$ A = (P, Q) $, $\tau = (cos\alpha , cos\beta)$为有向弧 $ L$在点$ (x, y)$处的单位切向量。$dr= \tau ds = (dx, dy)$称为**有向曲线元** 。 也可以记作: $$ \int_{L } A \cdot dr = \int_{L} A_{\tau} ds $$其中，$ A_{\tau}$称为向量$ A$在向量$ \tau$上的投影
