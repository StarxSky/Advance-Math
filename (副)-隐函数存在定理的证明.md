# 隐函数存在定理的证明

假设:函数满足以下几点:
1. 函数$ F\space $ 在以点 $P_{0}(x_{0}, \space  y_{0}) $为内点的某一区域$ D \sub R^{2}$ 内连续
2. $ F(x_{0}, \space y_{0}) = 0 $ (初值条件)
3. 函数$ F$在区域$ D$内具有连续的偏导数 $ F_{y}(x, \space y) = \frac{dF}{dy}$
4. $ F_{y}(x_{0}, \space y_{0})  = \frac{dF}{dy}|_{x=x_{0}}^{y=y_{0}} \ne 0 $

那么方程$ F(x, y) = 0 $ 在点$ P_{0}(x_{0}, \space y_{0})$ 的某个邻域$U(p_{0}) \sub D$内，唯一确定了一个定义在某个区间$(x_{0}- \alpha , \space x_{0} + \alpha)$上的函数$ y = f(x) $ (隐函数)。同时，其满足以下几点:
1. $y_{0} = f(x_{0}) ; \space x \in (x_{0}- \alpha , \space x_{0} + \alpha) $时， $ (x, \space f(x)) \sub D , \space F(x, \space f(x)) \equiv 0$
2. $f(x)$在区间$ (x_{0} - \alpha , \space x_{0} + \alpha) $ 内连续

### 1. 证明：先证明隐函数的存在和唯一性

由条件4:$ F_{y}(x_{0}, y_{0}) \ne 0$可知，我们可以设为：$$ F_{y}(x_{0}, \space y_{0}) > 0$$

同时，我们由于条件3可知，$ F_{y}(x, \space y) $ 在 区域$ D $ 内连续，结合连续函数的局部保号性，存在点$P_{0}$ 的某一闭的方邻域$$ [x_{0} - \beta, \space x_{0} + \beta ] \times [y_{0}-\beta , \space y_{0} + \beta ]$$，使得在其区域上每一点处都有$ F_{y}(x_{0},y_{0})>0$.
因而，对每个固定的$x\in[x_0-\beta,x_0+\beta]$，作为$y$的一元函数，必定在上**严格增且连续**.

结合初始条件 $ F(x_{0}, \space y_{0}) = 0 $可以得到：
$$F(x_{0}, \space y_{0} - \beta ) < 0 \space;\space F(x_{0}, \space y_{0}+\beta) >0 $$

再由$F$的连续性条件1.，又可知道$F(x,y_0-\beta)$ 与$ F(x,y_0+\beta)$ 在$[x_{0}-\beta, x_{0}+ \beta]$ 上也是连续的. 因此由区间的保号性可知，存在$ \alpha > 0 \space ; \space (\alpha \leq \beta)$，当$x\in(x_0-\alpha,x_0+\alpha)$时恒有:
$$ F(x_{0}, \space y_{0}- \beta) < 0 \space ; \space F(x_{0}, y+ \beta ) > 0$$

此时，我们的方邻域$$ [x_{0} - \beta, \space x_{0} + \beta ] \times [y_{0}-\beta , \space y_{0} + \beta ]$$构成一个上边为$A^{'}B^{'}$下边为$AB$的方形区域，在边$A^{'}B^{'}$上时，$F(x, \space y)$取正值，在边$AB $上时，$F(x, \space y)$取负值.同时，对于区间$ (x_{0} - \alpha, \space x_{0}+\alpha) $内的固定值 $ \bar{x} $也存在:$$ F(\bar{x}, \space y_{0} - \beta) < 0 \space ; \space F(\bar{x}, \space  y_{0} + \beta) > 0 $$
根据介值定理，可知，在区间 $ (\bar{x} - \alpha, \space \bar{x} + \alpha) $中存在唯一的一点$ \bar{y} \in (y_{0} - \beta, \space y_{0} + \beta) $使得 $ F(\bar{x}, \space \bar{y}) = 0$。同时，由于 $ \bar{x} $的任意性, 因此$ F(x, \space y) $ 唯一确定了一个函数 $ y =  f(x) $ 其定义域为: $ (x_{0}-\alpha , \space x_{0} + \alpha) $,值域为 $ (y_{0}- \beta , \space y_{0}+ \beta)$
