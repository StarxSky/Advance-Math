# 曲面的切平面与法线
## 隐式给出曲面方程的情况下
有曲面方程：$$F(x, \space y , \space z)$$ 
同时，在其上面存在一个空间曲线：$ \Gamma $其以参数方程形式给出：
$$ 
\left \{ 
    \begin{array}{lc}
        x = \phi(t) \\ 
        y = \mu(t) \\
        z = \omega(t)

    \end{array}
\right. ; \space (\alpha \leq t \leq \beta)
$$

参数 $ t = t_{0} $ 对应于点$ M(x_{0}, \space y_{0}, \space z_{0}) $且该三个函数的导数$ \phi(t)^{'} $ $\mu(t)^{'} $ $ \omega(t)^{'} $均不为零，得该曲线的切线方程为 $$ \frac{x - x_{0}}{\phi(t_{0})^{'}} =  \frac{y- y_{0}}{\mu(t_{0})^{'}} =  \frac{z - z_{0}}{\omega(t_{0})^{'}}$$
因为曲线$ \Gamma $完全在曲面 $ F(x, \space y, \space z) $ 上，故，有以下恒等式: $$ F[\phi(t), \space \mu(t), \space \omega(t)] \equiv 0 $$
此时，对该恒等式两边同时对变量$ t = t_{0} \space $求全导:
$$ \frac{d}{dt} F[\phi(t), \space \mu(t) , \space \omega(t)] \space |_{t = t_{0}}\equiv 0$$
$$ 
\frac{\partial{F}}{dx} \frac{dx}{dt} |_{t=t_{0}} + \frac{\partial{F}}{dy} \frac{dy}{dt} |_{t=t_{0}} + \frac{\partial{F}}{dz} \frac{dz}{dt} |_{t=t_{0}}= 0
$$
$$

\frac{\partial{F}}{dx} \phi(t_{0})^{'} + \frac{\partial{F}}{dy} \mu(t_{0})^{'} + \frac{\partial{F}}{dz} \omega(t_{0})^{'}= 0
$$

此时，我们结合前面提到的在点$ M(x_{0}, \space y_{0}, \space z_{0}) $处的切向量$$ T^{\rightarrow}(\phi(t_{0})^{'}, \space \mu(t_{0})^{'}, \space \omega(t_{0})^{'}) $$可知，式子$
\frac{\partial{F}}{dx} \phi(t_{0})^{'} + \frac{\partial{F}}{dy} \mu(t_{0})^{'} + \frac{\partial{F}}{dz} \omega(t_{0})^{'}= 0
$中分离出以下两个向量:
* 曲面切平面的法向量: $ n = (F_{x}(x_{0}, \space y_{0}, \space z_{0}), \space F_{y}(x_{0}, y_{0}, z_{0}), \space F_{y}(x_{0}, y_{0}, z_{0}) ) $

* 曲平面上的一个向量,(与向量$ n$垂直):$\space T^{\rightarrow}(\phi(t_{0})^{'}, \space \mu(t_{0})^{'}, \space \omega(t_{0})^{'}) $

## 显式 $ z = f(x, \space y ) $ 给出曲面方程的情况下

将所给的显式方程给转换为隐式形式：
$$ F(x, \space y, \space z) = z - f(x, \space y) = 0 $$
此时可以依照隐式情况进行分析得到曲面在该点$ M$处的法向量为：
$$ n = [f_{x}(x_{0}, \space y_{0}), \space f_{y}(x_{0}, \space y_{0}), \space -1]$$
同时得到在点$ M(x_{0}, \space y_{0}, \space z_{0})$处的切面方程为:
$$f_{x}(x_{0}, \space y_{0})(x-x_{0}) + f_{y}(x_{0}, \space y_{0})(y-y_{0}) + f_{z}(x_{0}, \space y_{0})(z-z_{0}) = 0 $$
