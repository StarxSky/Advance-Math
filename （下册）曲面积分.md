# 曲面积分

###  对面积的曲面积分

> 定义： 设曲面$ \Sigma$ 是光滑的，函数$ f(x, y, z)$ 在该曲面上有界，把$ \Sigma$ 分为任意$ n$个小的曲面$ \Delta S_{i}; \space i \in \{ 1, 2, \cdots, n\}$ ,(同时也代表着每个小曲面的面积为$ \Delta S_{i}$ )设 点$ (\sigma_{i}, \omega_{i}, \delta_{i})$为小曲面区域$ \Delta S_{i}$ 的一个点,做乘积$$ f(\sigma_{i}, \omega_{i}, \delta_{i}) \Delta S_{i}$$并作和式： $$ \sum_{i = 1}^{n}{f(\sigma_{i}, \omega_{i}, \delta_{i}) \Delta S_{i}}$$此时，如果各个小曲面$ \Delta S_{i} $的直径的最大值$ \lambda \to 0 $且上述和式的极限存在，且与点$(\sigma_{i}, \omega_{i}, \delta_{i})$的取法， 与小曲面的分发无关那么得到函数$ f(x, y,z)$ 在曲面$ \Sigma$ 上对曲面的面积的积分： $$ \iint_{\Sigma} {f(x, y,z)dS}  = \lim_{\lambda \to 0}{\sum_{i = 1}^{n}{f(\sigma_{i}, \omega_{i}, \delta_{i})\Delta S_{i}}}$$ 也叫**第一类曲面积分**。
* 性质： 
  1. 如果曲面$ \Sigma$是个分片光滑的，规定函数在$\Sigma$上的对面积的曲面积分等于函数在各个分片上的对面积的曲面积分的和。
  2. 对面积的曲面积分的性质与对弧长的曲线积分性质相似。

#### 对面积的曲面积分的计算方法

首先假设曲面: $$ \Sigma : z = z(x, y) $$来确定。曲面在平面$ xOy$上的投影为$ D_{xy}$，函数$ z = z(x, y)$投影区域$ D_{xy}$上具有连续的偏导数.且，被积函数$ f(x, y,z)$在曲面$ \Sigma$上面连续。

根据对面积的曲面积分的定义有： 
$$ \iint_{\Sigma} {f(x, y,z)dS}  = \lim_{\lambda \to 0}{\sum_{i = 1}^{n}{f(\sigma_{i}, \omega_{i}, \delta_{i})\Delta S_{i}}}$$
在该式子中：$$ \Delta S_{i} = \iint_{(\Delta S_{i})_{xy}}{\sqrt{1 + z_{x}^{2} + z_{y}^{2}}} dx dy$$
其中，$(\Delta S_{i})_{xy}$ 为小曲面$ \Delta S_{i}$ 在$ xOy$平面上的投影区域。
根据对二重积分的中值定理：
$$ \Delta S_{i}  = \\
\iint_{(\Delta S_{i})_{xy}} {\sqrt{1 + z_{x}^{2} + z_{y}^{2}} dxdy} = \sqrt{1 + z_{x}^{2} {(\mu_{i}^{'}, \epsilon_{i}^{'})}+ z_{y}^{2} {(\mu_{i}^{'}, \epsilon_{i}^{'})}} \iint_{(\Delta S_{i})_{xy}} dx dy \\ = \sqrt{1 + z_{x}^{2} {(\mu_{i}^{'}, \epsilon_{i}^{'})}+ z_{y}^{2} {(\mu_{i}^{'}, \epsilon_{i}^{'})}} \cdot (\Delta S_{i})_{xy}
$$
其中的点$ (\mu_{i}^{'}, \epsilon_{i}^{'})$为区域$ (\Delta S_{i} )_{xy}$上的一点。根据函数$ f[x, y, z(z,y)]$与$ \sqrt{1 + z_{z}^{2} + z_{y}^{2}}$的连续性可知:
$$
\lim_{\lambda \to 0} \sum_{i =1}^{n} {f(\sigma{i}, \omega_{i} , \delta_{i})}\Delta S_{i} = \lim_{\lambda \to 0} \sum_{i =1}^{n} {f(\sigma{i}, \omega_{i} , \delta_{i})} \sqrt{1 + z_{x}^{2} {(\mu_{i}^{'}, \epsilon_{i}^{'})}+ z_{y}^{2} {(\mu_{i}^{'}, \epsilon_{i}^{'})}} \cdot (\Delta S_{i})_{xy}  \\ = \lim_{\lambda \to 0}\sum_{i =1}^{n} {f(\sigma{i}, \omega_{i} , \delta_{i})} \sqrt{1 + z^{2}_{x} (\mu_{i} , \epsilon_{i}) + z^{2}_{y} (\mu_{i} , \epsilon_{i}) }\cdot (\Delta S_{i})_{xy}
$$

那么于是就有： $$ \iint_{\Sigma} {f(x, y,z)dS}  =\lim_{\lambda \to 0} \sum_{i =1}^{n} {f(\sigma{i}, \omega_{i} , \delta_{i})} \sqrt{1 + z^{2}_{x} (\mu_{i} , \epsilon_{i}) + z^{2}_{y} (\mu_{i} , \epsilon_{i}) }\cdot (\Delta S_{i})_{xy} \\ = \iint_{D_{xy}}f[x, y, z(x, y)] \sqrt{1 + z_{x}^{2} {(x, y)}  + z_{y}^{2} {(x, y)}  }dxdy$$
故，对于由方程$ z = z(x, y)$确定的曲面$ \Sigma$的对于面积的曲面积分的计算方法为$$ \boxed{\iint_{\Sigma }{f(x, y, z) dS} = \iint_{D_{xy}}f[x, y, z(x,y)] \sqrt{1 + z^{2}_{x} {(x, y)}  + z^{2}_{y} {(x, y)}  }dxdy} $$
* 由方程$ x = x(y,z)$确定的曲面$ \Sigma$的对于面积的曲面积分的计算方法为：$$ \boxed{\iint_{\Sigma} {f(x, y, z) dS} = \iint_{D_{yz}}f[x(y,z), y, z] \sqrt{1 + x^{2}_{y} {(y, z)}  + z^{2}_{z} {(y, z)}  }dydz} $$

### 对坐标的曲面积分

> 定义： 曲面$ \Sigma$为可分段光滑的**有向曲面** 【有向曲面是指可以通过曲面的法向量的方向来确定曲面的侧的曲面】。函数$ R(x, y, z)$在曲面$ \Sigma$上有界。将曲面分为$ n$块小曲面$ \Delta S_{i} \space ; \in \{ 1, 2,3, \cdots, n\}$（记每块小曲面的面积也为$ \Delta S_{i}$）,在平面$ xOy$上的投影区域为$ (\Delta S_{i})_{xy}$ （其面积也记作 $ (\Delta S_{i})_{xy}$），任取区域$ \Delta S_{i}$上的一个点 $(\sigma_{i}, \omega_{i}, \epsilon_{i}) \in \Delta S_{i}$。并做乘积求和： $$ \sum_{i = 1}^{n} { R(\sigma_{i}, \omega_{i}, \epsilon_{i})(\Delta S_{i})_{xy} } $$。如果在每个区域$ \Delta S_{i}$的直径的最大值$ \lambda \to 0$时，上式的极限存在，且与曲面的分法和点$ (\sigma_{i}, \omega_{i}, \epsilon_{i})$的取法无关，那么称之为**对坐标$ x, y$的曲面积分**记作：$$ \boxed{\iint_{\Sigma} R(x, y, z)dxdy = \lim_{\lambda \to 0} {\sum_{i = 1}^{n} { R(\sigma_{i}, \omega_{i}, \epsilon_{i}) (\Delta S_{i})_{xy}} }}$$
像类似的，有 **对坐标$ y, z$的曲面积分**：
>$$\boxed{ \iint_{\Sigma} P(x, y, z)dzdy = \lim_{\lambda \to 0} {\sum_{i = 1}^{n} { P(\sigma_{i}, \omega_{i}, \epsilon_{i}) (\Delta S_{i})_{yz}} }}$$
>**对坐标$ z, x$的曲面积分**
>$$\boxed{ \iint_{\Sigma} Q(x, y, z)dzdx = \lim_{\lambda \to 0} {\sum_{i = 1}^{n} { Q(\sigma_{i}, \omega_{i}, \epsilon_{i}) (\Delta S_{i})_{xz}} }}$$
>以上三个曲面积分也称为**第二类曲面积分**.
* 性质： 
  1. 如果曲面$ \Sigma$是个分片光滑的，规定函数在$\Sigma$上的对坐标的曲面积分等于函数在各个分片上的对坐标的曲面积分的和。
  2. 对坐标的曲面积分的性质与对坐标的曲线积分性质相似。

#### 对坐标曲面积分的计算方法

* 对于曲面$\Sigma$由方程$ z = z(x, y)$来确定的。曲面在平面$ xOy$上的投影为$ D_{xy}$，函数$ z = z(x, y)$在投影区域$ D_{xy}$上具有连续的偏导数.且，被积函数$ R(x, y,z)$在曲面$ \Sigma$上面连续。
    1) 当取曲面$\Sigma$的上侧时： 按照对坐标的曲面积分的定义有$$ \iint_{\Sigma} R(x, y, z)dxdy = \lim_{\lambda \to 0} {\sum_{i = 1}^{n} { R(\sigma_{i}, \omega_{i}, \epsilon_{i}) (\Delta S_{i})_{xy}} }$$因为取的是曲面$\Sigma$的上侧，所以$ cos \gamma > 0$并且根据平面的投影公式$ \frac{dA}{cos \gamma} = d\sigma$有： $$  \Delta S_{i} \cdot cos \gamma  = (\Delta S_{i})_{xy}>0 $$
    又因为点$ (\sigma_{i}, \omega_{i}, \epsilon_{i}) \in \Delta S_{i}$，因此有：$ \epsilon_{i} = z(\sigma_{i}, \omega_{i})$此时： $$ \lim_{\lambda \to 0 } \sum_{i = 1}^{n} { R(\sigma_{i}, \omega_{i}, \epsilon_{i}) (\Delta S_{i})_{xy}} = \lim_{\lambda \to 0 } \sum_{i = 1}^{n} { R[\sigma_{i}, \omega_{i}, z(\sigma_{i}, \omega_{i})] (\Delta S_{i})_{xy}}$$此时另各个曲面$\Delta S_{i}$的直径最大者$ \lambda \to 0$有：$$ \lim_{\lambda \to 0} \sum_{i = 1}^{n} { R[\sigma_{i}, \omega_{i}, z(\sigma_{i}, \omega_{i})] (\Delta S_{i})_{xy} } = \iint_{D_{xy}} { R[x, y, z(x, y)] dxdy}$$
    此时，得到对曲面$ \Sigma$的上侧的对坐标$ x, y$的曲面积分的计算公式：$$ \boxed{\iint_{\Sigma} R(x, y, z)dxdy = \iint_{D_{xy}} { R[x, y, z(x, y)] dxdy}} $$

    2) 当取得曲面$ \Sigma$的下侧的时候有：$$ \boxed{\iint_{\Sigma} R(x, y, z)dxdy = - \iint_{D_{xy}} { R[x, y, z(x, y)] dxdy}} $$

* 对于由方程$ x = x(y, z)$确定的曲面$ \Sigma$的对于坐标$y, z$的曲面积分的计算方法为：$$ \boxed{\iint_{\Sigma} P(x, y, z)dydz = \pm \iint_{D_{yz}} { P[ x(y, z), y, z] dydz}} $$

* 对于由方程$ y = y(x, z)$确定的曲面$ \Sigma$的对于坐标$z, x$的曲面积分的计算方法为：$$ \boxed{\iint_{\Sigma} Q(x, y, z)dzdx = \pm \iint_{D_{xz}} { Q[z, y(x, z), x] dzdx}} $$


### 两类曲面积分之间的联系

首先给出有关两类曲面积分之间的联系的结论：$$ \boxed{\iint_{\Sigma} P(x, y, z) dydz + Q(x, y, z) dzdx + R(x, y, z)dxdy \\ = \iint_{\Sigma} {[Pcos\alpha + Qcos\beta + Rcos\gamma ]} dS  }$$
其中，其中，$ n = (cos \alpha , cos \beta, cos \gamma)$为曲面$ \Sigma$的单位法向量。同时，$ cos \alpha, cos \beta, cos \gamma$分别为曲面$ \Sigma$的法向量的方向余弦: $$ \boxed{cos \alpha = \frac{-\frac{\partial{z}}{\partial{x}}}{\sqrt{1 + \left(\frac{\partial{z}}{\partial{x}}\right)^{2} + \left(\frac{\partial{z}}{\partial{y}}\right)^{2}}} ;\\ \space \\ cos \beta = \frac{-\frac{\partial{z}}{\partial{y}}}{\sqrt{1 + \left(\frac{\partial{z}}{\partial{x}}\right)^{2} + \left(\frac{\partial{z}}{\partial{y}}\right)^{2}}} ;\\ \space \\  cos \gamma = \frac{1}{\sqrt{1 + \left(\frac{\partial{z}}{\partial{x}}\right)^{2} + \left(\frac{\partial{z}}{\partial{y}}\right)^{2}}} }$$同时，该式子也可以记作向量形式：$$ \iint_{\Sigma} {A \cdot dS} = \iint_{\Sigma} {A \cdot ndS} $$其中$ A = (P, Q, R)$ 并且$ n = (cos \alpha, cos \beta , cos \gamma)$
