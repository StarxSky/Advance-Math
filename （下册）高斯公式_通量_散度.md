# 高斯公式
> 定义： 空间区域$ \Omega$由分片光滑的曲面$ \Sigma$围成，若函数$ P(x, y,z), Q(x,y,z), R(x, y, z)$在区域$ \Omega$上具有连续的一阶偏导数，则有：$$ \boxed{\iiint_{\Omega} (\frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}}   ) dv = \oiint_{\Sigma} (Pdydz + Qpdxdz + Rdxdy)  \\ \space \\ = \oiint_{\Sigma}{ (Pcos\alpha + Qcos \beta + Rcos \gamma) dS }}$$这里的$ \Sigma$是$ \Omega$的整个外侧。同时，$$ cos \alpha, cos \beta, cos \gamma$$为$ \Sigma$在点$ (x, y,z)$的方向余弦。并且：$$cos \alpha = \frac{-z_{x}}{\sqrt{1 + z_{x}^{2} + z_{y}^{2}}};\space cos \alpha = \frac{-z_{y}}{\sqrt{1 + z_{x}^{2} + z_{y}^{2}}} ;\space cos \alpha = \frac{1}{\sqrt{1 + z_{x}^{2} + z_{y}^{2}}} $$
> 注意⚠️：（高斯公式**仅限于在闭区域**上有效，如果区域非闭合区域则不能保证公式的有效性）

# *格林第一公式
> 函数$ u = u(x, y, z)$与函数$ v = v(x, y,z)$在闭区域$ \Omega$的整个边界曲面,$ \frac{\partial{v}}{\partial{n}}$为函数$ v(x, y,z)$ 沿着$ \Sigma$的外法向量方向的方向导数。那么：$$ \iiint_{\Omega}{u\Delta v dxdydz} = \oiint_{\Omega} {u \frac{\partial{v}}{\partial{n}} dS} - \iiint_{\Omega}{(\frac{\partial{u}}{\partial{x}} \frac{\partial{v}}{\partial{x}} + \frac{\partial{u}}{\partial{y}} \frac{\partial{v}}{\partial{y}} + \frac{\partial{u}}{\partial{z}} \frac{\partial{v}}{\partial{z}}) dxdydz}$$
这个公式叫做**格林第一公式**

# 通量

> 定义： 首先假设存在一个向量场： $$ A(x, y,z) = P(x, y, z) i + Q(x, y,z) j + R(x, y,z) k$$其中的函数$ P, Q, R$均在定义域上存在连续的一阶偏导数，$ \Sigma$为向量场中的一块有向曲面。$ n$为该有向曲面的单位法向量，那么有**通量**： $$ \iint_{\Sigma} {\bar{A} \cdot \bar{n} dS}$$.同时根据两类曲面积分之间的关系有： $$ \boxed{\iint_{\Sigma} {\bar{A} \cdot \bar{n} dS} = \iint_{\Sigma}{\bar{A} \cdot \bar{dS}} = \iint_{\Sigma}Pdydz + Q dzdx + Rdxdy}$$

# 散度
> 由高斯公式可得到：$$ \iiint_{\Omega} {(\frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}} ) dv} = \oiint_{\Sigma}{Pdydz + Qdzdx + Rdxdy}$$.现在以区域$ \Omega$的体积$ V$除以上式两端得到：$$ \frac{1}{V} \iiint_{\Omega} {(\frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}} ) dv} = \frac{1}{V} \oiint_{\Sigma}{Pdydz + Qdzdx + Rdxdy} $$现在对上式左端进行积分中值定理得到： $$ \frac{1}{V}  V (\frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}})|_{(\epsilon, \mu, \delta)} = \frac{1}{V} \oiint_{\Sigma}{Pdydz + Qdzdx + Rdxdy}$$这里的点$ (\epsilon, \mu, \delta)$为区域$ \Omega$中的一点，令该点缩向点$ M(x, y,z)$取上式的极限：$$ \frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}} = \lim_{\Omega \to M} \frac{1}{V} \oiint_{\Sigma}{Pdydz + Qdzdx + Rdxdy}$$其中，上式左端部分称为点$ M$的**散度/通量密度**：$$ \boxed{\mathbf{div}{(M)} = \frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}}}$$同时有以下几种情况： 
> 1） 当散度$ \mathbf{div} (M) > 0$ 时：表示流体从该点处向外部发散，表示流体在该点处具有**正源**； 2） 当散度$ \mathbf{div}{(M)} < 0$时：表示流体向该点处聚集，表示流体在该点处具有**负源**又称为“洞”；3）当散度为零时 $ \mathbf{div} (M) =0$ :流体在该点处**无源**。

对于一般的向量场$$ A(x, y,z) = P(x, y,z)i + Q(x, y,z)j + R(x, y,z)k$$其散度为： $$ \mathbf{div} (M) = \frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}} $$，如果在该向量场中，散度处处为零，那么该向量场称为**无源场**。利用微分算子记作： $$ \nabla \cdot A = \frac{\partial{P}}{\partial{x}} + \frac{\partial{Q}}{\partial{y}} + \frac{\partial{R}}{\partial{z}} $$那么高斯公式可以记作以下的向量形式： $$ \boxed{\iiint_{\Omega} {\mathbf{div} (A)}  dv = \iiint_{\Omega} \nabla\cdot A dv= \oiint_{\Sigma} Pdydz + Qdzdx + Rdxdy  = \oiint_{\Sigma} A \cdot n dS= \oiint_{\Sigma} A_{n} dS}$$
上式说明：高斯公式表示的是向量场$ A$通过闭曲面$ \Sigma$流向外侧的通量等于向量场的散度$ \mathbf{div}(A)$在闭曲面$ \Sigma$所围成的区域内$ \Omega$的积分
